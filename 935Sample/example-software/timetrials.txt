Here when we run time trials we can log the times it took

Time Trial Instructions:
    - Start the timer as soon as the sync changes button is clicked
    - When the first log appears lap the timer
    - Stop the timer the moment this message is logged:
        - [Watcher] Deployment completed successfully.

Time Trial Key:

    - TTL = time to start logs
    - TTF = time to finish
    - EXC = Extreme Conditions (New packages to install ect.)
    - TLD = Difference between the start logs time from the past trials
    - TFD = Difference between the finsih time from the past trial
