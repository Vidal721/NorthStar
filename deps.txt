# NorthStar Project Dependencies
# Generated for a brand new system setup

## 1. Core Software & Programs
- **Node.js LTS** (Recommended: Version 20.x or newer)
- **Python 3.10+** (System includes Python 3 by default on most Linux distros)
- **Java 17 JDK** (REQUIRED for Android builds; Oracle JDK or Eclipse Temurin)
- **Android Studio** (For building the Scouter Tablets app)
- **Android SDK & Build Tools** (API Level 33 specifically mentioned)
- **ADB (Android Debug Bridge)** (Part of `android-tools` or `platform-tools`)
- **Git** (For version control)
- **Code OSS / VS Code** (Recommended IDE)
- **Tesseract OCR Engine** (System binary required for `pytesseract`)

## 2. System Dependencies (Linux/Arch/Ubuntu)
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install nodejs npm python3-pip adb tesseract-ocr libtesseract-dev libgl1-mesa-glx libbluetooth-dev bluez bluetooth python3-tk

# Arch Linux
sudo pacman -S nodejs npm python-pip android-tools tesseract tesseract-data-eng libbluetooth bluez bluez-utils tk
```

## 3. Python Libraries (via pip)
Install these for the vision and server components:
```bash
pip install opencv-python numpy ultralytics filterpy flask scipy paramiko pytesseract pybluez
```
*Note: `pybluez` may require additional setup for Bluetooth compatibility mode.*
